ЧТО ПЕРЕВЕДЕНО
  • Весь текст демо: диалоги, квесты, записки и документы,
    описания предметов и оружия, радиопереговоры, меню,
    настройки, титры, подсказки на загрузочных экранах.
  • Шрифты заменены на кириллические аналоги в стилистике игры
    (Bebas Neue Cyrillic, RUS BOYCOTT).
  • НЕ переведено: субтитры видеовставок

УСТАНОВКА (автоматическая)
  1. Запустите install.bat
  2. Скрипт сам найдёт папку игры и скопирует файл русификатора.

УСТАНОВКА (вручную)
  Скопируйте файл AbyssGame-Windows_RU_P.pak в папку:
    ...\Steam\steamapps\common\The Path Into The Abyss Demo\
        Windows\AbyssGame\Content\Paks\
  Рядом с AbyssGame-Windows.pak. Больше ничего делать не нужно.
  Язык в настройках игры должен быть English (по умолчанию).

УДАЛЕНИЕ
  Запустите uninstall.bat или просто удалите файл
  AbyssGame-Windows_RU_P.pak из папки Paks.
  Оригинальные файлы игры не изменяются.

ВАЖНО
  • Русификатор сделан для публичной пре-альфа демоверсии
    (сборка июля 2026). После обновления демо перевод может
    частично перестать работать до выхода новой версии.
  • Это неофициальный фанатский перевод, не связанный
    с разработчиками игры.

БЛАГОДАРНОСТИ
  • Разработчикам игры — за атмосферный аргентинский
    постапокалипсис. Изначально в игре только английский
    и испанский; этот патч добавляет русский.
  • Шрифты: Bebas Neue Cyrillic, RUS BOYCOTT (свободные
    кириллические версии), Roboto (Apache License 2.0).

  Перевод: yaruncle + Claude
